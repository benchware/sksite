Request centre

Start a request, report an issue, request a record or check the status of something you already submitted.

How requests work

  1. Choose the request type below.
  2. Submit the form and save the reference number.
  3. Use Check request status with your reference number and email address.
  4. If a record is approved for release, the access instructions appear in the status result.
